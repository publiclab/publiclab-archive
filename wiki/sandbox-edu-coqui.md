# sandbox-edu-coqui

by [mimiss](../profile/mimiss), [warren](../profile/warren)

May 23, 2019 18:43 | Tags: 

----

The Coqui: a simple device to read a sensor via an audible tone. We're exploring using audio frequencies to convey sensor readings -- like a modem, but even simpler -- in order to make sensor readings more accessible for people (or to enable data transmission over ubiquitous audio jacks on PCs and smartphones)

For example, a Coqui can be built to measure:

 - [Conductivity](https://publiclab.org/notes/donblair/09-30-2014/coqui-bbv1-0-testing-conductivity-of-a-solution) (this capacity included in Coqui Classroom Kits)
 - [Temperature](https://publiclab.org/notes/donblair/09-30-2014/coqui-bbv1-0-using-a-thermistor-as-a-temperature-probe)
 - [Light](https://publiclab.org/notes/donblair/09-30-2014/coqui-bbv1-0-assessing-ambient-light-with-a-photoresistor)
