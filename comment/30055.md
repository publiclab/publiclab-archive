Hello [@stefq1111](/profile/stefq1111) thank you for your proposal! It's looking great. I like the detailed thought you've put into things like where "help" links go, and the layout and sequencing of the page, and use of icons in limited space. A small note is that it looks like your page layout has vertical scrolling and I wonder if there's a benefit to having vertical full-screen as well, so an edge-to-edge vertical interface which is always 100% height? I know the current design doesn't have this but consider a page like https://photopea.com/ where it's more like using a desktop app.

It'll be interesting to see if you can get the horizontal toolbar to reflow vertically, or if you'll have to have 2 different toolbars, with CSS or JS causing only one to be shown. Do you think there could be a steeper learning curve to have 2 different layouts depending on screen size, or do you think the benefits of the vertical layout (which i agree make more sense for step-by-step layout of tools) outweigh the risks?
