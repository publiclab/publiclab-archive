This is a test of Public Lab's MapKnitter online program to 'stitch' microscope imagery (instead of map features).  The image is a 13 picture mosaic of a honeybee leg.  ( For  more info on resolution: 
 https://publiclab.org/notes/MaggPi/06-23-2018/microscope-calibration-overlays-what-s-the-best-approach.  or   https://publiclab.org/notes/MaggPi/06-05-2018/usb-webcam-vs-raspberry-pi-v2-field-of-view-fov-comparison-community-microscope-initial-evaluation-2   )  


 Scroll in/out to see the different 'leg' features!

The test demonstrates  how MapKnitter can scale/project other pictures beside aerial photos.  

If anyone knows how to start with a grid or blank background (instead of a map)  please make a comment.....
