# Arduino Uno Thermo Flashlight Casing Specs

The following document briefly summarizes and details the casing specs for the Arduino Thermo Flashlight for MUN's Sociology 4107 class. It was built using a CNC router with scrap wood collected from a local hardware store. Total time for the casing estimates about 8 hours (mostly due to time spent cutting on the router itself).
