# Google Summer of Code 2013 & Public Lab

by [gonzoearth](/profile/gonzoearth) | April 17, 2013 18:22

Public Lab is happy to announce that we are part of the Google Summer of Code (GSOC) 2013 program!

This program introduces students from colleges and universities around the world to open source software development through a stipend for summer coding on selected mentored projects. 

We need your help in developing the projects and recruiting students!

More information about the program here:
http://publiclab.org/wiki/gsoc-ideas

Help formulate, promote, or apply for Public Lab projects here: http://publiclab.org/wiki/gsoc-2013
