# Picavet PET rig

by [mathew](/profile/mathew) | February 06, 2012 02:58

February 06, 2012 02:58 | Tags: [activity:photo-rig](/tag/activity:photo-rig)

----

[A Picavet rig](http://www.kaper.us/basics/BASICS_picavet.html) is a very common type of KAP/BAP rig, I've added it to the juice bottle in a fairly simple way, using 1/2" key rings.

<a href="https://www.flickr.com/photos/14397636@N07/6827181665/" title="Picavet Juice bottle by mathew.lippincott, on Flickr"><img src="https://farm8.staticflickr.com/7030/6827181665_068e676a92.jpg" width="375" height="500" alt="Picavet Juice bottle"></a>

It makes a good Zip line attachment, and looks like it might provide better stability, especially in light wind, than the [single line + fin system we've been using](http://youtu.be/Oi63BGmgsRA).
