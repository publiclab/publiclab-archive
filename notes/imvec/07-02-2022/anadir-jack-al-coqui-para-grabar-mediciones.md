# Añadir clavija al sensor Coqui para grabar las mediciones

### Lo que queremos hacer

Añadir una clavija minijack al sensor Coqui par registrar las mediciones sonoras del sensor Coqui directamente con una grabadora de audio o una computadora conectándolo con un cable.

Con esta modificación también es posible amplificar la señal del Coqui utilizando unos altavoces.

### Cómo añadir la clavija

Primero sigue el impresionante post de @warren traducido por @alejobonifacio para [construir tu propio sensor de electroconductividad Coqui](https://publiclab.org/notes/alejobonifacio/05-10-2022/construya-un-sensor-de-conductividad-coqui-generador-de-sonido) y luego continúa leyendo estas instrucciones.

Hemos utilizado la clavija "mini jack" [Thonkiconn 3.5mm](https://www.thonk.co.uk/shop/3-5mm-jacks/ "https://www.thonk.co.uk/shop/3-5mm-jacks/") que puedes encontrar en internet y hemos soldado dos cables para facilitar su introducción en la protoboard.

[![thonkiconn.jpg](/i/25180)](/i/25180)

### Enchufa los cables como se enchufa el altavoz:

\***Cable rojo** al Pin#3  
\***Cable verde** a tierra![image description](/i/46841.png "COQUI_.png")
